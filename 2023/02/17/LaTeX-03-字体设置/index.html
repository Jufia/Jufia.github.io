LaTeX(03)-字体设置

发表于 2023-02-17 分类于 Language ， LaTeX

¶摘要

字体、页码、页面

¶字体的包

  • 正常字体：\mathnormal{} --> normal
  • 罗马体：\mathrm{} --> roman
  • 斜体：\mathit{} --> italic
  • 黑粗体：\mathbf{} --> boldface
  • 黑板体（空心体）：\mathbb{} --> blackboard
  • 书法艺术体：\mathcal{} --> calligraphy
  • 书写体(圆体)：\mathscr{} --> script
  • 德文尖角体（哥特体）：\mathfrak{} --> fraktur (aka gothic)
  • 无衬线体：\mathsf{} --> sans serif
  • 打字机体：\mathtt{} --> typewriter type

一般用fontspec包来设置字体

字体大小

¶页码、页眉、页脚设置

可以用\pagestyle{style}设置每页的页眉（\header）和页脚(\footer)的设置，style的可选项是：

  1. \plain：缺省，页码打印在页脚中间；
  2. \headings：页码，每章标题打印在页眉，页脚留空；
  3. \empty：设置页眉页脚为空。

对特定页可以用：\ thispagestyle{style}可以设置 独立的页码页眉设置。

¶atex中采用的单位

  1. pt：一个点point相当于1/72.27inch，也就是大约0.0138inch或0.3515mm ；
  2. mm：毫米； cm：厘米； in：英寸；
  3. ex：约等于当前字体中字母x（小写）的高度；
  4. em：约等于当前字体中字母M（大写）的宽度；
  5. mu：当前采用的数学字体为参考的em，则1mu=1/18em。
