数学相关

极值理论 EVT

在自然界中的许多现象都服从着高斯分布，例如大部分人的身高都离平均身高不远，而高斯分布偏离中心的的窄而长的两端的“尾巴”表示有的人特别高，有的人特别矮。高斯分布的“尾巴”是不好预测的小概率事件，想要预测小概率事件的发生概率只要使用以往该小概率事件发生情况的拟合曲线即可。EVT的中心思想是概率分布，即给出事件发生概率的数学公式，EVT有着健全的数学基础而不仅仅是经验公式。
个人图书馆-极值理论Extreme Value Theory，解释了EVT是什么以及它的作用。
CSDN-EVT 极值理论

韦布尔分布 Weibull Distribution

一种数据分布方式，韦伯分布被经常用来对失效性（time to Failure）或者，反而言之为，可靠性，进行衡量的工具。
使用方法：设定一个阈值 threshold，将需要判断的点x与平均数的距离带入到累计分布函数中，得到的值大于 threshold 则说明该点异常（故障）。(见论文X. Yu et al., “Deep-Learning-Based Open Set Fault Diagnosis by Extreme Value Theory,” in IEEE Transactions on Industrial Informatics, vol. 18, no. 1, pp. 185-196, Jan. 2022, 中 Algorithm 2)
