微积分-2解微分方程

发表于 2022-07-13 更新于 2022-07-24 分类于 math ， 微积分

常系数齐次线性微分方程

微分方程形式：

\begin{equation}
y^{(n)} + a_1 y^{(n-1)} + \dots + a_n y = 0
\end{equation}
y的解的形如：

\begin{equation}
y = e^{\lambda x}
\end{equation}
带入上式，解得：

\begin{equation}
\lambda ^{n} + a_1 \lambda ^{n-1} + \dots + a_n = 0
\end{equation}
即为上述微分方程的特征方程，其根为特征根。

  • n个互异的特征根$\lambda_1 \dots \lambda_n$ ,对应着$e^{\lambda_1 x}， e^{\lambda_2 x}, \dots , e^{\lambda_n x}$ 为一个基本解组;
    若有复特征根，则必成对出现：$\lambda_k = \alpha \pm i \beta$,
    $y_1 = e^{\alpha x}cos\beta x$
    $y_2 = e^{\alpha x}sin\beta x$
  • 若有重根
    若$\lambda_i$ 为$k$重根，对应着：$e^{\lambda _i x}, xe^{\lambda _i x}, \dots , x^{k-}1e^{\lambda _i x}$

常系数非齐次线性微分方程

微分方程：
\begin{align}
y^{(n)} + a_1 y^{(n-1)} + \dots + a_n y = f(x)
\end{align}
微分方程的特征方程:
\begin{equation}
\lambda ^{n} + a_1 \lambda ^{n-1} + \dots + a_n = 0
\end{equation}

解微分方程的通解

$$\lambda_k = \alpha + i \beta$$

$$
\begin{array}{rl}
\begin{cases}
y_1 &= e^{\alpha}cos\beta x \cr
y_2 &= e^{\alpha}sin\beta x
\end{cases}
\end{array}
$$
微分方程的解存在且有唯一性。

解微分方程的特解

微分方程等号右边的非齐次项形如：
$$f(x) = e^{\alpha x}[P(x)cos \beta x + Q(x)sin \beta x]$$
$P(x)、Q(x)$是多项式，$m$是它们的次数的最大值，若$\alpha + i\beta$是特征方程的$k$重特征根，则非齐次方程的特解形如：
$$y_* = x^k e^{\alpha x}[R(x)cos \beta x+ Q(x)sin\beta x]$$
$S(x)、R(x)$是$x$的$m$次待定多项式，带入原微分方程中可求解$R(x)、Q(x)$的系数。最终是通解为：
$$y = y + y_*$$
