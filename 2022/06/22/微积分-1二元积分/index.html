格林公式

格林定理给出了沿封闭曲线 C 的线积分与以 C 为边界的平面区域 D 上的双重积分的联系。格林定理是斯托克斯定理的二维特例。
设闭区域 D 由分段光滑的简单曲线 L 围成，函数 P(x,y)及 Q(x,y)在 D 上有一阶连续偏导数，则有：
$$\oint _{L^+}(P;dx + Q;dy) = \iint (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y});dxdy $$
其中闭环曲线L逆时针旋转时取正。它给出了沿着闭曲线C的曲线积分与C所包围的区域D上的二重积分之间的关系。另见格林恒等式。格林公式还可以用来计算平面图形的面积。

斯托克斯定理

散度 $div {\quad} \overrightarrow{v}$

散度的物理意义表示流体微团在在运动过程中体积相对变化率的极限。
将各速度分量在其分量方向上的方向导数和称为速度矢量的散度
即： $\nabla \cdot \overrightarrow{v} = div \overrightarrow{v} = \frac{\partial v_x}{\partial x} + \frac{\partial v_y}{\partial y} + \frac{\partial v_z}{\partial z}$

旋度

$$
\begin{array}{rl}
\nabla \times \overrightarrow{v} &= rot \overrightarrow{v} \cr
& = \iint_{S}\left(\frac{\partial R}{\partial y}-\frac{\partial Q}{\partial z}\right)dydz+\left(\frac{\partial P}{\partial z}-\frac{\partial R}{\partial x}\right)dzdx+\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)dxdy \cr
& =\oint _{\Gamma }Pdx+Qdy+Rdz
\end{array}
$$

斯托克斯定理

$$
\begin{array}{l}
\iint_{S}\left(\frac{\partial R}{\partial y}-\frac{\partial Q}{\partial z}\right)dydz+\left(\frac{\partial P}{\partial z}-\frac{\partial R}{\partial x}\right)dzdx+\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)dxdy \cr
= \oint _{\Gamma }Pdx+Qdy+Rdz
\end{array}
$$

曲线积分

第一类曲线积分

是对一段曲线（弧）$L$的积分。积分微元为弧微分$ds$，被积函数由于在弧线上取值，因此是二元函数$f(x, y)$.积分值记为$\int_L f(x, y) ds$.
其中被积函数$f(x, y)$在$L$上取值。
弧$L$上的点满足的方程可以由两种形式给出，一种是$y = y(x)$,另一种则是参数方程的形式$x = \phi (t), y = \psi (t)$
对于第一种形式，被积函数可以写成只和$x$有关的一元函数，即$f(x, y(x))$，弧微分可以写成$ds = \sqrt{(dx)^2 + (dy)^2 = \sqrt{1 + y’(x)^2}dx}$,
则曲线积分化为一重积分$\int_{x_1}^{x_2} f(x, y(x)) \sqrt{1 + y’(x)^2}dx$.
对于第二种形式，被积函数可以写成只和$t$有关的一元函数，即$f(\phi (t),\psi (t))$ ，弧微分可以写成$ds = \sqrt{(dx)^2+ (dy)^2 } \sqrt{\phi’(t)^2 + \psi’(t)^2}dt$
这就是第一类曲线积分，可用于求解不均匀曲线段的质量，被积函数即为线质量密度。

第二类曲线积分

解微分方程

可分离变量类型$\frac{dy}{dx} = \frac{f(x)}{g(y)}$

$g(y)dy = f(x)dx$
$\int g(y) ,dy = \int f(x) ,dx + C$

齐次方程$\frac{dy}{dx} = f(\frac{y}{x})$

等式左边为导数，右边为$f(\frac{y}{x})$的函数，这也是很特殊的形式。
令$u(x) = \frac{y(x)}{x}$，则$y(x) = x u(x)$。
由乘积求导法则，有$\frac{dy(x)}{dx} = u(x) + x \frac{du(x)}{dx} = f(u(x))$。
$\frac{du(x)}{dx} = \frac{f(u(x)) - u(x)}{x}$
$\Rightarrow \int ,du= \int \frac{f(u(x)) - u(x)}{x},dx$
由$y(x) = xu(x)$即可得到$y(x)$

一阶常微分方程$\frac{dy}{dx} + P(x)y = Q(x)$

$y = e^{-\int P,dx}[ \int e^{\int P,dx}Q(x) ,dx + C ]$
